SDSU Marathon 2016

San diego's biggest Marathon is back!

Gear up and run for charity!

The SDSU Marathon will host 3 race categories – The full marathon (20 miles), half marathon (10 miles) and the ever popular 5 mile run (5 miles) race. All the runners will be given a goody bag with a running T-shirt, running bib (with timing chip), finisher’s medal and a breakfast tuck box.


FULL MARATHON - 20 miles
The race of the event will be run on a certified course which will take you through SDSU through west till Point Loma. Expect a flat, speedy route and an early start.


HALF MARATHON - 10 miles
The happy middle ground for runners who have run multiple 5 miles already but are still some time away from planning a full marathon. The race route will be a certified course and take runners past many of the sights and sounds which the full marathoners enjoy.


5 mile run
The ever popular 5 miles event is a runaway success every year. Expect to see thousands of enthusiastic, excited, first time runners across age categories rub shoulders with focused, super fast, experienced athletes at the start line. If you’ve never run before, this should be your go-to event of the year.
